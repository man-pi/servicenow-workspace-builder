# ServiceNow Workspace Builder Skill

A Claude skill for building custom ServiceNow Next Experience workspaces programmatically via REST API and GlideRecord scripts.

## What This Covers

- The full 6-record chain required to render a workspace
- REST API patterns for creating workspace components
- Scheduled job workaround for ACL-blocked sys_ux tables
- Troubleshooting blank workspace content
- Reverse-engineering working workspaces (like CMDB) as templates

## Files

- `SKILL.md` - Main skill instructions and workflow
- `references/record-chain.md` - Detailed reference with composition JSON, script templates, and API examples
